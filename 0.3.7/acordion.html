Acordeón Básico

¿Qué es lucen UI?

lucen UI es un framework basado en Bootstrap que proporciona componentes con estilo minimalista, usando solo HTML, CSS y JS.

¿Cómo se usa?

Simplemente incluye los archivos CSS y JS de lucen UI en tu proyecto. Todos los componentes están listos para usar con las clases proporcionadas.
