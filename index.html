¿Qué es Lucen UI?

Lucen es un framework de interfaz de usuario minimalista que proporciona componentes y herramientas para construir aplicaciones web modernas con un diseño limpio y consistente.

Incluye componentes personalizables, utilidades CSS y patrones de diseño listos para usar en tus proyectos.
