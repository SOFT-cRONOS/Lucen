Lucen UI Framework

Diseño minimalista para interfaces web

¿Qué es Lucen?

Lucen es un framework de interfaz de usuario minimalista que proporciona componentes y herramientas para construir aplicaciones web modernas.

Componentes

Botones, tarjetas, formularios y más

Estilos

Variables CSS y clases utilitarias

Rendimiento

Ligero y optimizado

¿Dónde encontrar lo que necesito?

Usa el menú lateral para navegar entre las secciones
  • Componentes básicos: En la sección "Interface" encontrarás botones, tarjetas, formularios y más
  • Personalización: En "Utilities" puedes modificar colores, apariencia y clases
  • Elementos completos: En "Web Elements" hay componentes como headers, footers y formularios de contacto
  • Documentación: Cada página incluye ejemplos y guías de uso
